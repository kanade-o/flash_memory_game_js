1S No.6 Sota Ohnuma

Flash Memory Game

Current level: 0
Description

概要: 光ったブロックの順番にクリックしていくゲームです.

操作方法: GAME STARTでスタート. ブロックにカーソルを合わせてクリック.

<工夫事項>

  • setTimeoutを用いて, ブロックの光る時間を調整した.
  • CSSでhoverやtransaction, ブロックの配置をgridを使って見た目に拘った.
  • 配列を使ったり, html部分ではなくjs側で動きを加えることで可読性を向上させた.

<苦心事項>

  • クリックしたら配列に入れて, 答え合わせをするという流れで混乱した.
  • DOMが正しく取得されないときがあったが, htmlに書くscriptの位置を変えることで解決した.
